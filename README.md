# Camvital Scan (iOS)

This repository contains the **native iOS app** only (SwiftUI, HealthKit, camera PPG).

## Open in Xcode

1. Open **`CamvitalScan/CamvitalScan.xcodeproj`**
2. Select the **CamvitalScan** scheme and your development team for signing.

Details, requirements, and feature notes are in **`CamvitalScan/README.md`**.
